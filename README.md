### How to run

- Using [Docker](https://docs.docker.com/engine/installation/):

  docker build -t nickollascoelho/cinq-angularjs .
  docker run -p 49876:3000 -d nickollascoelho/cinq-angularjs

  Go to [localhost:49876](http://localhost:49876)
